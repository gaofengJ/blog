## 为什么学可视化

* 第一，现在很多 C 端或者 B 端的互联网产品离不开可视化

* 可视化能实现很多传统 Web 网页无法实现的效果

* 可视化可以利用 WebGL 这样的图形系统，更高效地实现更多有趣的图形

## 可视化工程师的要求

<span style="color: #FA8919;">一名优秀的可视化工程师，最应当具备的能力就是要根据产品经理和设计师的想法，实现真正贴合用户需求的、灵活多变的、具有视觉震撼力，并且充满创意的优秀可视化作品。</span>
