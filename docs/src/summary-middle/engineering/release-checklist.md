# 生产发布 checklist 写点啥

## 发布内容

* 项目名称
* 更新内容
* 代码合并情况
* 测试环境 / pre 环境发布情况
* 负责人

## 配置变更

* 域名配置
* 路径配置
* 中台配置

## 发布流程

* 发布流程
* 回滚方案

## 检查项

### 发布前

* 流程工单
* 中台接入情况
* 公共插件接入情况
* CR 情况
* 权限相关
* 特殊功能点
* 发布日程

### 发布后

* 功能验证
* 信息同步
* 插件、中台接入验证
* 分支合并
* 特殊功能点
